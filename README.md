# 💕 Happy Anniversary Nanu & Nani Website

A beautiful, romantic anniversary website with animated elements and a photo gallery!

## Features

- 🎨 Beautiful gradient background with romantic color scheme
- 💫 Smooth animations and hover effects
- 📸 Responsive photo gallery with 6 image slots
- 📱 Fully responsive - works on desktop, tablet, and mobile
- 🎭 Glassmorphism design elements
- ❤️ Animated hearts

## How to Use

1. **Open the website**: Simply open `index.html` in your web browser
2. **Replace photos**: 
   - Replace the image URLs in the `<img>` tags with your own photos
   - Currently uses placeholder images from Unsplash
   - Update the `src` attribute in each gallery item
   
   Example:
   ```html
   <img src="YOUR_IMAGE_URL_HERE" alt="Memory">
   ```

3. **Customize text**:
   - Edit the `index.html` file to change any text
   - Modify colors in `styles.css` by changing the gradient colors or hex values

## Customization Ideas

### Change Colors
Open `styles.css` and modify:
- Line 12: `background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);` - Change to your preferred colors

### Add Your Photos
Replace the image URLs in `index.html` with:
- Direct image file paths: `src="photos/photo1.jpg"`
- Online URLs: `src="https://example.com/photo.jpg"`

### Change Text
- Line 2 (index.html): Change the title
- Line 11: Change "Nanu & Nani" to your names
- Line 60 onwards: Customize memory labels

## Tips

- Use high-quality landscape images for best results
- Keep image sizes consistent (square aspect ratio works best)
- Test on mobile devices for responsive design
- Use a local web server for best performance with local images
